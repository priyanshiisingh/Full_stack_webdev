pointer for block

  1. width of a block element is 100% of the viewport
  2. height of a block isn't set by default
  3. height and width can be explicitly set using css

pointer for inline elements

  1. width of a inline element is dependent on content
  2. height of a inline isn't set by default
  3. height and width can be explicitly set using css
  4. cannot put block elements inside inline element
